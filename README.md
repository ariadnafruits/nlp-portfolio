# NLP Portfolio

This repository brings together a collection of **Natural Language Processing (NLP)** projects developed during a Master's degree in Language Technologies.

The portfolio reflects a progression from **rule-based linguistic systems** to **statistical and semantic models**, culminating in applied machine learning in speech processing.

---

## 🧩 Projects

### 🔹 Spanish Morphological Analyzer
Development of a **rule-based morphological analyzer for Spanish**, combining linguistic rules and lexical resources.

- Regular expression–based morphological patterns  
- Dictionary-based tagging  
- Linguistically motivated rule design for POS assignment  

📁 Repository: https://github.com/youruser/spanish-morphological-analyzer

---

### 🔹 Feature-Based Grammar for Spanish Command Recognition
Rule-based system for interpreting **robot commands in Spanish** using a **feature-based context-free grammar (FCFG)** with semantic parsing.

- Syntactic parsing with feature structures  
- Semantic representation via attribute-value structures  
- Integration of syntax and semantics through **feature unification**  

📁 Repository: https://github.com/youruser/spanish-command-recognition

---

### 🔹 Statistical POS Tagger Comparison
Evaluation of different statistical part-of-speech taggers using the **Brown corpus**.

- Trigrams’n Tags (TnT)  
- Standford MaxEnt POS Tagger

### 🔹 Vector Space Model Representations
Implementation of document representation techniques within the **Vector Space Model (VSM)** framework.

- Text preprocessing and normalization  
- Term weighting schemes (TF, IDF and variants)  
- Construction and comparison of document vectors  

📁 Repository: https://github.com/youruser/vector-space-model-representations

---

### 🔹 TextRank Keyword Extraction
Graph-based keyword extraction using a custom implementation of the **TextRank algorithm** on POS-tagged corpora.

- Construction of co-occurrence graphs  
- Application of ranking algorithms inspired by PageRank  
- Extraction of document-level keywords based on graph centrality  

📁 Repository: https://github.com/youruser/textrank-keyword-extraction

---

### 🔹 Semantic Textual Similarity
Implementation and comparison of multiple approaches to measure semantic similarity between sentence pairs using the **STS Benchmark dataset**.

- Cosine similarity over semantic vector representations
- Custom pairwise similarity aggregation  
- Overlap-based approach 

📁 Repository: https://github.com/youruser/semantic-textual-similarity

---

### 🔹 Prosody-Based Curriculum Learning for ASR (master thesis )
Analysis of **prosodic features** as predictors of transcription difficulty in automatic speech recognition (ASR) systems, and their use to design **curriculum learning strategies**.

- Study of the relationship between prosody and Word Error Rate (WER)  
- Evaluation on feature impact on model performance  
- Application to training strategies for **Transformer-based ASR systems**  

📁 Repository: https://github.com/youruser/prosody-based-curriculum-learning-asr
